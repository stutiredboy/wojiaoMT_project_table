help			显示帮助信息
------------------------------------------------------------------------------
load			载入序列化文件
	参数：jobj_path
	jobj_path	- 序列化文件的路径
------------------------------------------------------------------------------
show			显示当前已经载入的序列化文件
------------------------------------------------------------------------------
use				设置当前的序列化对象
	参数：jobj_name
	jobj_name	- 序列化对象的名字
------------------------------------------------------------------------------
operate			执行获取值和设置值的操作
	参数：exp
	exp			- 获取值和设置值的表达式
------------------------------------------------------------------------------
save			将当前对象序列化到文件中
	参数：path
	path		- 文件路径
------------------------------------------------------------------------------
exit			退出客户端
